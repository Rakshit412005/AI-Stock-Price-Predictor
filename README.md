# AI Stock Price Predictor

An AI-powered Stock Price Prediction web application built with Streamlit.

## Features

- Live stock data using Yahoo Finance
- Multiple Machine Learning models
- News sentiment analysis
- Investment simulation
- Interactive charts
- Model comparison

## Tech Stack

- Python
- Streamlit
- Scikit-learn
- Plotly
- Yahoo Finance API
- VaderSentiment
- TextBlob

## Run Locally

```bash
pip install -r requirements.txt
streamlit run stock_predictor_app.py
```
